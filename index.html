How to Play Game ?


1. First Create Username.


2. click Start.


3. The Aim is to click each card on the screen.


4. You'll be shown a color , each color has a partner which is identical.


5. The aim is to get the lowest number of attempts/time to match all the pairs.


6. The first card flip will only match with the second card flip and so on.


7. There are 16 Cards in total which means there are 8 Pairs to be matched


8. Most importantly Enjoy & try with friends.
